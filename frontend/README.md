# Frontend - Meu CRM

Frontend simples em HTML + Tailwind (CDN) + JavaScript puro.

## Páginas

- `login.html`
- `dashboard.html`
- `clientes.html`
- `processos.html`
- `atividades.html`
- `documentos.html`

## Execução

- Acesse `http://localhost:3000` para o frontend servido pelo backend.
- Alternativamente, abra `pages/login.html` com Live Server.
